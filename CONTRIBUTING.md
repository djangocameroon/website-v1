# Guide de Contribution pour le site Django Cameroon

Nous sommes ravis que vous souhaitiez contribuer à notre projet ! Ce document vous guidera à travers le processus.

## Comment commencer

1.  **Forkez le projet** : Cliquez sur le bouton "Fork" en haut à droite de la page GitHub.
2.  **Clonez votre fork** : `git clone https://github.com/VOTRE_NOM_UTILISATEUR/website-v1.git`
3.  **Naviguez vers le répertoire du projet** : `cd website-v1`
4.  **Installez les dépendances** :
    ```bash
    npm install
    ```

## Lancer le projet en développement

Pour voir vos modifications en direct, lancez le serveur de développement :

```bash
npm start
```

Cela ouvrira le site sur `http://localhost:3000`.

## Lancer les tests

Assurez-vous que vos modifications ne cassent rien en lançant la suite de tests :

```bash
npm test
```

## Soumettre vos contributions

1.  **Créez une nouvelle branche** pour vos modifications : `git checkout -b ma-nouvelle-fonctionnalite`
2.  **Faites vos modifications** et commitez-les avec un message clair.
3.  **Poussez votre branche** vers votre fork : `git push origin ma-nouvelle-fonctionnalite`
4.  **Ouvrez une Pull Request** sur le dépôt original. Décrivez les changements que vous avez faits.

Merci pour votre contribution !
